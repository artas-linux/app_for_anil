Delivery Policy (Digital Services)

Effective Date: October 8, 2025

Website and Application Delivery

Websites and applications are delivered via a staging server or source code repository. Acceptance occurs upon User Acceptance Testing (UAT) sign-off by the client.

Software Licenses

Software licenses are provisioned within 24-72 hours after payment confirmation. Clients will receive access credentials via the registered email address.

Training Services

Training sessions are delivered on scheduled dates as shared via email invitation or calendar invite. Materials are provided in digital format via email or shared drive.

Documentation

Service documentation, including technical specifications and user manuals, is delivered via email or through a shared repository. Links to documentation are provided in the project completion email.

Source Code Delivery

Upon project completion and final payment, source code is delivered through secure repository access or encrypted file transfer as per client preference.

Delivery Timeline

Delivery timelines are specified in the original project agreement. Standard delivery times are:

  • Starter Website: 2 weeks
  • E-Commerce Launch: 4-6 weeks
  • Custom API Integration: 3-4 weeks
  • Android/iOS App (MVP): 8-10 weeks
  • Consulting Projects: As per agreed scope

Delivery Confirmation

Clients will receive an email notification upon delivery with instructions for accessing the delivered service. All deliveries require client acknowledgment to complete the project cycle.
